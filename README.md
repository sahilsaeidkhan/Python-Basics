# Python-Basics

This repository contains beginner-friendly Python scripts and exercises to help you learn and practice fundamental programming concepts. It is organized into folders covering different topics, with clear examples and exercises.

## Structure

- **first.py**: Entry-level script for basic Python syntax.
- **Codebasics - tutorials/**: Contains scripts for various Python concepts:
	- `dummy.py`, `forloop.py`, `function.py`, `if_conditinal.py`, `Input.py`, `Lists.py`, `Lists2.py`, `numbers.py`, `String.py`, `tempCodeRunnerFile.py`, `Variables-Exercise.py`
- **Functions/**: Focused on functions and arguments:
	- `print.py`, `withargu.py`

## How to Use

1. Clone or download the repository.
2. Open any script in your preferred editor (e.g., VS Code).
3. Run scripts using Python 3.x:
	 ```
	 python <script_name.py>
	 ```
4. Explore, modify, and experiment with the code to reinforce your learning.

## Topics Covered

- Variables and data types
- Input/output operations
- Conditional statements
- Loops (for, while)
- Functions and arguments
- Lists and string operations
